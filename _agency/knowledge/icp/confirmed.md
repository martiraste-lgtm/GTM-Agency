# ICP Confermati

*Pattern promossi da hypotheses.md dopo 3+ conferme in campagne diverse.*
*Aggiornare tramite `skills/brain-update/`.*

---

## Come leggere questo file

Ogni pattern confermato ha:
- **Segnale**: quale trigger ha generato il risultato
- **Contesto**: in quale situazione ha funzionato
- **Risultato**: metrica osservata (reply rate, tier conversione, ecc.)
- **Note**: dettagli utili per replicare

---

*Nessun pattern confermato ancora. Il sistema è nuovo — i pattern si accumulano con l'uso.*

---

## Template per nuovo pattern confermato

```
### [CODICE SEGNALE] — [Nome segnale]
**Confermato**: [N] volte | **Prima conferma**: [DATA] | **Ultima**: [DATA]
**Contesto**: [settore / fase / persona target]
**Risultato**: reply rate [X%] vs baseline [Y%]
**Note**: [cosa ha funzionato esattamente — copy angle, timing, canale]
```
